# AI ChatBot using Java

A simple AI ChatBot developed using Java Swing and basic NLP concepts for interactive communication.

## 🚀 Features

* Simple and clean GUI
* Real-time chatting
* Rule-based AI responses
* Beginner-friendly chatbot logic
* Java Swing interface
* Easy to understand and modify

## 🛠 Technologies Used

* Java
* Swing
* OOP Concepts
* Event Handling
* Basic NLP Logic

## 📌 How to Run

1. Open project in Eclipse or VS Code
2. Create a Java class named `AIChatBot`
3. Paste the code
4. Run as Java Application

## 💡 Sample Questions

* Hello
* How are you
* What is AI
* What is Java
* Bye

## 📚 Learning Outcome

This project helped in understanding:

* Java GUI Development
* Event Handling
* Chatbot Logic
* Object-Oriented Programming
* Basic Artificial Intelligence Concepts

## 🔮 Future Improvements

* Voice Assistant

* Dark Mode UI
